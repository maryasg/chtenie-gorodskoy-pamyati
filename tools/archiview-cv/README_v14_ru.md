# Archiview CV v14

Версия v14 улучшает вкладки **«Разметка»** и **«Результат»** для работы с крупными фасадами.

## Запуск

```bat
install_windows.bat
run_gui_windows.bat
```

## Новое в v14

### Разметка и результат
- **Колесо мыши** — zoom (точка под курсором).
- **Пробел + ЛКМ** — временная «рука», перемещение полотна (как в Photoshop).
- **Средняя / правая кнопка** — перемещение (как в v13).

### Результат
- Такой же zoom и панорамирование, что в разметке.
- **Номера областей** — без чёрной подложки, жирным, в цвете типа изменения.
- **Наведение на область** — всплывающая подсказка (номер, тип, комментарий).
- **Список легенды** — наведение подсвечивает область; клик центрирует вид.

### Файлы
- `07_marked_on_original_modern.png` — фасад с областями и лёгкими номерами.
- `08_marked_on_original_modern_labeled.png` — фасад сверху, легенда снизу (без текста поверх фото).

### Индикатор
- При несохранённых изменениях в статусе разметки: «● Есть несохранённые изменения».

## Workflow (без изменений)

Источники → Выпрямление → Сравнение → Разметка → Результат.

## Связь с сайтом «Память стен»

После сохранения скопируйте в проект сайта:
- `07_marked_on_original_modern.png`
