**Папка программы:**
`archiview_cv_easy_v15_package`

Старые версии (v11–v14 внутри v13) можно не трогать — работайте только из **v15**.

## Запуск

```bat
install_windows.bat
run_gui_windows.bat
```

## Где лежит результат

После «Сохранить разметку» откройте папку проекта, вкладка 5 или кнопка «Открыть папку результата».

Пример:
`archiview_projects\20260520_190036\result\`

## Папка `archiview_projects` — откуда берутся данные

**Важно:** в `archiview_projects` не нужно вручную складывать все архивные фото. Туда попадает только **рабочая пара**, которую вы выбрали в программе.

### Как это работает

1. Запустите Archiview → вкладка **«1. Выбор фото»**.
2. **Историческое фото:** поиск в PastVu по адресу **или** кнопка «Файл с компьютера» (ваш скан, фото из архива).
3. **Современное фото:** Wikimedia / OpenStreetCam / Mapillary **или** «Файл с компьютера» (ваша полевая съёмка).
4. Программа **сама копирует** выбранные файлы в папку проекта:
   - `historical_sources/` — выбранное историческое
   - `modern_sources/` — выбранное современное
   - `result/` — выпрямленные фото, разметка, JSON

### Почему папки называются по-разному

| Имя папки | Откуда |
|-----------|--------|
| `new_house_project` | Стартовая папка при первом запуске |
| `Чистопрудный_бульвар_14` (или похожее) | Программа создаёт по **адресу** дома |
| `20260520_190036` | Старый формат: дата+время создания проекта |
| Имя файла современного фото | Если адрес не введён — по имени вашего файла |

**Вы не создаёте эти папки вручную.** Можно нажать «Папка…» и указать своё имя (например `MOSCOW_003`), но проще ввести адрес — программа предложит понятное имя.

### Что куда класть (и что не класть)

| Действие | Куда |
|----------|------|
| Весь архив PastVu, все ваши снимки | **Отдельно на диске** (не в archiview_projects) |
| Одна пара «история + сегодня» для работы | **Выбор во вкладке 1** → программа скопирует сама |
| Готовый результат для сайта | `copy_to_website.bat` → `public/explorer/MOSCOW_003/` |

### Для второго, третьего, четвёртого здания

1. Вкладка 1 → введите **новый адрес** (или «Папка…» → новая папка, например `MOSCOW_001`).
2. Выберите историческое и современное фото.
3. Выпрямление → разметка → `copy_to_website.bat` (нужно будет указать ID здания в bat/ps1).
4. На сайте: `public/explorer/<ID>/` — запись в `archiviewAssets.ts` **обновляется автоматически** при `copy_to_website.bat` (см. `website_buildings.json`).

Подробнее: `КАК_РАБОТАЕТ_ПАПКА_ПРОЕКТА_ru.md` в этой же папке.

Перетаскивание вершин пока нет — только удалить и перерисовать. Если криво на сайте — проверьте 4 угла во вкладке 2 и заново сохраните разметку.

## Как отправить на сайт (GitHub)

1. Запустите **`copy_to_website.bat`** в этой папке v15 (копирует в проект сайта автоматически).
2. Откройте **GitHub Desktop** → репозиторий `chtenie-gorodskoy-pamyati`.
3. Увидите изменения в `public/explorer/MOSCOW_003/` → **Commit** → **Push**.
4. Через 2–3 мин сайт обновится: https://maryasg.github.io/chtenie-gorodskoy-pamyati/explorer

Ручной путь описан в `КАК_ОТПРАВИТЬ_НА_САЙТ.txt`.
