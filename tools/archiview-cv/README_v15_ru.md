# Archiview CV v15 (отдельная папка)

**Папка программы:**
`archiview_cv_easy_v15_package`

Старые версии (v11–v14 внутри v13) можно не трогать — работайте только из **v15**.

## Запуск

```bat
install_windows.bat
run_gui_windows.bat
```

## Новое в v15

- Курсор **рука** (hand2) при пробеле и при перетаскивании полотна.
- Список **«Области — выберите номер и удалите»** — удаление любой области, не только последней.
- Всё из v14: zoom, tooltip на результате, лёгкие номера.

## Где лежит результат

После «Сохранить разметку» откройте папку проекта, вкладка 5 или кнопка «Открыть папку результата».

Пример:
`archiview_projects\20260520_190036\result\`

Главные файлы:
| Файл в result | Зачем |
|---------------|--------|
| `07_marked_on_original_modern.png` | Фасад с номерами → на сайт |
| `08_marked_on_original_modern_labeled.png` | Фасад + легенда снизу |
| `annotations\manual_annotations.json` | Подписи и полигоны |

## Как отправить на сайт (GitHub)

1. Запустите **`copy_to_website.bat`** в этой папке v15 (копирует в проект сайта автоматически).
2. Откройте **GitHub Desktop** → репозиторий `chtenie-gorodskoy-pamyati`.
3. Увидите изменения в `public/explorer/MOSCOW_003/` → **Commit** → **Push**.
4. Через 2–3 мин сайт обновится: https://maryasg.github.io/chtenie-gorodskoy-pamyati/explorer

Ручной путь описан в `КАК_ОТПРАВИТЬ_НА_САЙТ.txt`.
