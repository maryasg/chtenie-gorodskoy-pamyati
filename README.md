# Чтение городской памяти через фасад

Пилотный демо-MVP: карта 4 зданий Москвы, карточки со следами памяти, экскурсия, режим исследователя, симуляция AR.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Результат в `dist/`. Для GitHub Pages задан `base: /chtenie-gorodskoy-pamyati/`.

## Структура

- `src/data/buildings/` — данные пилотных зданий (MOSCOW_001–004)
- `content/` — YAML для редакции (по мере наполнения)
- `src/pages/` — карта, экскурсия, исследователь, карточка
